| Имя поля ввода       | Описание                                                                                                                           | Регулярное выражение |
|----------------------|------------------------------------------------------------------------------------------------------------------------------------|----------------------|
| `login`              | Любое сочетание латинского алфавита с арабскими цифрами и символами '_', '-' и '.'; Минимальная длина 4 символа, максимальная - 16. Уникальное значение. | `^[-a-zA-Z._\\d]*$`   |
| `nickname`           | Любое сочетание символов UNICOD; Минимальная длина 4 символа, максимальная - 16. Уникальное значение.                                                   | -                    |
| `password`           | Любое сочетание символов UNICOD; Минимальная длина 8 символа, максимальная - 256                                                   | `/^(?=.*\d)(?=.*[A-Z])(?=.*[a-z])(?=.*[^\w\d\s:])([^\s]){8,256}$/gm`                 |
| `password_confirmed` | Должно совпадать с `password`                                                                                                      | -                    |
